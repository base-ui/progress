
# progress

  Base progress bar

## Installation

  Install with [component(1)](http://component.io):

    $ component install base-ui/progress

## Quick Start
```javascript
var Progress = require('progress');
var progress = new Progress($('.ui.progress'));
```

## API
### progress
#### .percentage
#### .setPercentage(percentage)
#### .on('percentage', callback)
